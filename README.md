# ψ-GLIF

Public publication archive, provenance registry, intellectual property framework, and smart archival tooling for ψ-GLIF works.

## Repository Scope

This repository contains intentionally approved public materials.

It is not a mirror of PSI-LAB and does not contain private development infrastructure, internal runtime information, unpublished economic systems, wallet data, credentials, or automatic synchronization mechanisms.

## Core Areas

- Public publication records
- Provenance documentation
- Intellectual property preparation
- Archival standards
- Smart archive tooling
- Public archive agents

## Publication Principle

Private creation remains private.

Nothing enters this repository automatically. Every artifact requires human review and explicit publication approval.
